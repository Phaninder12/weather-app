# Weather App

**Project Title:** Weather App  
**Technologies:** HTML, CSS, JavaScript  
**Difficulty Level:** Hard  
**API Used:** OpenWeatherMap (Free Tier)

---

## Project Description

A **responsive, user-friendly web-based weather application** that allows users to enter a city name and instantly view the **current weather conditions**, including:

- **Temperature** (in Celsius)
- **Weather Description** (e.g., "Sunny", "Cloudy", "Light Rain")
- **Weather Icon** (visual representation)

Built with **vanilla HTML, CSS, and JavaScript**, this project demonstrates **API integration**, **asynchronous programming**, **error handling**, and **responsive design**.

## How to Run the App

> **Important**: Do **not** open `index.html` directly. Use a **local server** to avoid CORS errors.

### Recommended Method: **VS Code + Live Server**

1. Install [VS Code](https://code.visualstudio.com/)
2. Install **"Live Server"** extension
3. Open project folder
4. Right-click `index.html` → **"Open with Live Server"**
5. App runs at: `http://127.0.0.1:5500`
